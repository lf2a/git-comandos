# Configuração de usuário

## Nome de usuário
```bash
$ git config --global user.name "Luiz Filipy"
```

## Email do usuário
```bash
$ git config --global user.email "luizfilipy.aa@outlook.com"
```

## Listando as configurações do git
```bash
$ git config --list
```

# Historico dos commits

```bash
$ git log --oneline
```
